📖 使用說明

瀏覽 我的 測驗 說明

🚀 30 秒上手

  1. 到 測驗 → 選「中翻英」 → 開始答題
  2. 答錯的字自動加 ⭐，答對 3 次的字自動標 ✓ 已掌握
  3. 明天打開「我的」 → 點「用 ⭐ 字測驗」就能複習弱點
  4. 每天 30 分鐘，3 個月後雅思 6.5 不是問題

📑 三個頁面在幹嘛

📘 瀏覽

看全部 1132 個單字，可以搜尋、按主題/難度/狀態篩選。

  • 點單字 = 念出來（英式發音）
  • 點例句旁的 🔊 = 念整句
  • ⭐ 按鈕 = 自己標記不熟的字
  • ✓ 按鈕 = 標記已掌握的字（之後不會再考你）
  • 翻面模式（右上勾選）= 中文跟例句先模糊，看英文先想，點才翻面

💡 用來：背新字、確認某個字的意思、找特定主題

⭐ 我的單字

只看你標記的字。三個 tab：

  • ⭐ 標記中：答錯或自己標的字（要複習的）
  • ✅ 已掌握：連續答對 3 次的字
  • 🆕 最近練過：所有你碰過的字

有兩個快速按鈕：

  • 📝 用 ⭐ 字測驗：一鍵跳到測驗頁，只考你的弱點字
  • 清空 ⭐ 標記：把所有 ⭐ 清掉（已掌握的不會被清）

💡 用來：每天 5–10 分鐘的快速複習

📝 測驗

三種模式，對應雅思三種技能：

  • 🎧 聽寫：聽發音 → 輸入單字。練聽力。
  • 📝 例句填空：看上下文 → 4 選 1。練閱讀。
  • 🔤 中翻英：看中文 → 輸入英文。練口說/寫作。

四個篩選器可以組合：

  • 題數：10/20/50/全部
  • 範圍：全部 / 只考 ⭐ / 只考未掌握
  • 目的：不限 / 口說寫作主動 / 閱讀聽力辨識
  • 類別：依主題（環境、科技、AWL 子表 1–10 等）

⭐ vs ✓ 是什麼

標記什麼時候出現意思
⭐ 標記 答錯時自動加；也可以手動點 ⭐ 按鈕 「這個字我還不熟，要再複習」
✓ 已掌握 連續答對 3 次自動加；也可手動點 ✓ 「這個字我會了，從測驗池子退役」

💡 設定「範圍 = 只考 ⭐」或「只考未掌握」可以更聚焦練習。

💾 我的進度會不見嗎？

正常使用不會。進度（⭐、✓、打卡、設定）存在瀏覽器的 localStorage，會一直在。

但有幾種情況會清掉：

  • 清除瀏覽器資料／快取／網站資料
  • 切到無痕模式
  • 換不同瀏覽器或裝置（進度不會跨裝置同步）
  • 在我的單字頁點「清空 ⭐ 標記」

💡 重要：所以同一個裝置一直用就好，別在電腦練了又在手機練。選一個。

✍️ 拼字答題的規則

系統會自動接受這些變化（不會誤判為錯）：

  • UK / US 拼字：colour = color，analyse = analyze，centre = center
  • 複數：skill / skills 都對
  • 動詞變化：study / studies / studied / studying 都對
  • 大小寫不分

但是：

  • 同義字不會被接受（例如 word 是 lamp，你寫 light 會被判錯）
  • 多字片語要寫完整（"look forward to" 不能只寫 "look forward"）

⌨️ 快捷鍵（電腦）

  • Enter：送出答案 / 進入下一題
  • 在聽寫題目：點 🔊 重播發音
  • 在瀏覽頁：點單字本身就會念

📥 想加新單字怎麼辦

你練雅思題目時遇到新字，跟你的 AI 助理（宗澄助理）說一聲，AI 會：

  1. 把新字加進「雅思準備/單字表.md」
  2. 同步加進這個網站的資料庫
  3. 自動 push 上線

1–2 分鐘後刷新網頁就能看到新字。

❓ 常見問題

為什麼點單字沒有發音？

瀏覽器第一次載入需要幾秒鐘準備語音引擎。等個 3 秒再點就好。如果還是不行：

  • 確認瀏覽器音量沒靜音
  • Safari 跟 Chrome 都支援；Firefox 可能要手動開啟
  • iOS 第一次要點任意一個按鈕「啟用」語音
明明答對為什麼判錯？

多半是這幾種情況：

  • 多打/少打了空格
  • 用了同義字（例如 word 是 lamps，你寫 lights）
  • 不規則變化（例如 word 是 run，你寫 ran），系統的寬容只蓋複數跟規則變化

如果你確定是 bug，跟 AI 說一聲幫你修。

練到一半要關掉怎麼辦？

按右上角的「退出」就可以。會出現確認彈窗，按確定即可。

注意：當次測驗的對錯數字會丟失，但個別單字的 ⭐ / ✓ 標記會保留。

已掌握 ✓ 的字想拿回來複習？

去「我的」→ 點「✅ 已掌握」tab → 找到那個字 → 點 ✓ 按鈕取消勾選，它就回到一般池子。

連續天數怎麼算？

每天任意答一題就算當天有打卡。中間漏一天就會歸零重新算。

1132 字會不會太少？

對雅思 6.5–7 分來說夠。AWL 570 個學術字是雅思閱讀核心，加上主題詞庫已涵蓋所有常見話題。

研究顯示 Band 7 候選人實際主動詞彙約 2800 個，1132 個高品質字 + 你練習累積的 = 綽綽有餘。

主動產出（active） vs 辨識（passive） 是什麼？

active = 自己要會寫會說的字（口說、寫作）。

passive = 看到聽到能反應出意思的字（閱讀、聽力）。

絕大多數英文母語者也只能主動用幾千個字，但能認得幾萬個。你不用每個字都能主動用。
